◆ VDP_REG00

VDP_REG00   0x8000

Mode Register 1.

7 6 5 4 3 2 1 0
L IE1 CM M3 DE
Parameters
LBlank leftmost 8 pixels to background color
IE1Enable horizontal interrupts
CM1 = normal 9-bit color mode (512 colour); 0 = low 3-bit color mode (8 colors)
M31 = freeze H/V counter on level 2 interrupt; 0 = enable H/V counter
DE0 = enable display; 1 = disable display

Mode Register 2.

7 6 5 4 3 2 1 0
VR DE IE0 M1 M2 M5
Parameters
VREnable 128kB of VRAM (incompatible with standard consoles)
DE1 = enable display; 0 = fill display with background colour
IE0Enable vertical interrupts (VINT)
M11 = enable DMA operations; 0 = ignore DMA operations
M21 = 240 pixel (30 cell) PAL mode; 0 = 224 pixel (28 cell) NTSC mode
M51 = Mega Drive (mode 5) display; 0 = Master System (mode 4) display

Parameters
WBits 15-11 (W5 to W1) of window nametable address in VRAM
Note

W6 is used with 128k VRAM only
