#define VDP_REG00   0x8000

Mode Register 1.

| 7| 6| 5| 4| 3| 2| 1| 0| |—+—+—+--—+—+-—+-—+-—| | 0 | 0 | L | IE1 | 0 | CM | M3 | DE |

L: 1 = leftmost 8 pixels are blanked to background color IE1: 1 = enable horizontal interrupts CM: 1 = normal 9-bit color mode (512 colour) 0 = low 3-bit color mode (8 colors) M3: 1 = freeze H/V counter on level 2 interrupt 0 = enable H/V counter DE: 1 = disable display

Mode Register 2.

| 7| 6| 5| 4| 3| 2| 1| 0| |-—+-—+--—+-—+-—+-—+—+—| | VR | DE | IE0 | M1 | M2 | M5 | 0 | 0 |

VR: 1 = use 128kB of VRAM; not compatible with standard consoles with 64kB VRAM DE: 1 = enable display 0 = fill display with background colour IE0: 1 = enable vertical interrupts M1: 1 = enable DMA operations 0 = ignore DMA operations M2: 1 = 240 pixel (30 cell) PAL mode 0 = 224 pixel (28 cell) NTSC mode M5: 1 = Mega Drive (mode 5) display 0 = Master System (mode 4) display

Mode Register 3.

| 7| 6| 5| 4| 3| 2| 1| 0| |—+—+—+—+--—+-—+--—+--—| | 0 | 0 | 0 | 0 | IE2 | VS | HS1 | HS2 |

IE2: 1 = enable external interrupts VS: Vertical scrolling mode 1 = 16 pixel columns (1 word per column in VSRAM) 0 = full screen (1 longword only in VSRAM). HS1-HS2: Horizontal scrolling mode 00 = full screen 01 = invalid 10 = 8 pixel rows 11 = single pixel rows.
